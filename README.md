# Student Result Management App

Project kun app xiqqoo ta'ee, barattoota, qopheessitoota, fi bu'aa isaanii sirnaan bulchuuf sirna salphaa dha.  
App Kivy fayyadamuun GUI gaarii fi sirrii qaba.

---

### **Features:**
- Barattoota galchuu, sirreessuu, fi haquu
- Qopheessitoota galchuu, sirreessuu, fi haquu
- Qorannoo fi bu'aa galchuu
- PDF Certificate fi Result generate gochuu
- User friendly UI fi workflow salphaa

---

### **Akkaataa hojjachuu:**
1. Dependencies install gochuu: `pip install -r requirements.txt`
2. App hojjachuu: `python main.py`
3. GUI irratti hojjetuu

---

### **Galmee:**
- `main.py`: Main application
- `main.kv`: UI design
- `student.py`: Student module
- `admin.py`: Admin module
- `exam.py`: Exam module
- `pdf.py`: PDF functions
- `buildozer.spec`: Build configuration
- `build.yml`: Workflow
- `requirements.txt`: Dependencies
